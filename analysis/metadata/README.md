# Metadata

This folder is intentionally kept out of Git except for this README.

Place local metadata files here when running the analysis scripts, for example:
- `N240_BMI.csv`
- `N240_age.csv`
- `N240_contrast.csv`
- `N240_gender.csv`
- `b66_split_maria.csv`

These files may contain sensitive or project-specific information and should not be pushed to GitHub.
